1.Bootstrap到底是什么？

（1）它是一个框架
（2）它用来开发web应用程序和网站前端
（3）它基于html css js

2.下载使用bootstrap

（1）下载地址 http://getbootstrap.com/
（2）有两种：1.预编译的 2.源代码 （这里我们使用预编译的）
（3）代码结构目录（js css fonts）

3.html中使用bootstrap

4.html中注意事项

（1）head中引入bootstrap的css路径
（2）bootstrap会用到jquery，需要引入jquery的js库路径
（3）在body标签的最后引入（运行速度更快）
